RESTGears by abossard

Easy, clean and simple RESTful backend for everyone

RESTGears tries to solve the need for RESTful Service in a generic way: - you can create a login - you can create several applications - each application has documents (key-value stores) and files - various tools help you to cope with special cases (resizing images, ...) - a sophisticated tagging and categories scheme, help you find your documents.

Dependencies

see requirements.txt

Install

pip install -r requirements.txt python manage.py syncdb python manage.py runserver
